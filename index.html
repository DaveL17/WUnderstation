About

The WUnderstation plugin allows users to upload Personal Weather Station (PWS) weather data to Weather Underground using their Indigo Home Control Server.

Installation

Installation is easy. Download and extract the Indigo Plugin file onto your Indigo server machine. Double-click the plugin file and follow the prompts. Presently, the plugin does not make use of any Indigo devices; all controls and data sources are managed through the plugin configuration dialog. The plugin supports only one PWS location at a time; the operation of the plugin is very simple.

  1. Within the plugin preferences, enter the PWS information and password for your PWS account.
  2. Enter the frequency of updates (in seconds)
  3. Select the data sources that you want to use for each Weather Underground data element--there are about 50 to choose from.

It is important to note that you are responsible for the data you upload to Weather Underground. For example, it's possible that you could report the brightness of a light as the outside temperature. So be careful and choose wisely.

Menu Items

There are a couple Indigo menu items associated with this plugin.

  • Check for Plugin Updates... Select this menu item to check to see if there is a new version of the plugin available. In order for this feature to function properly you must configure plugin update notifications within the plugin configuration dialog window.
  • Toggle Debugging Selecting this item will turn debugging mode on or off depending on its current setting. Additional debugging settings can be found in the plugin configuration dialog window.
